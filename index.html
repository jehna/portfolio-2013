Penetration

Penetration is a browser game made in 48 hours. The target is to protect your planet from forthcoming asteroids. The game was made in Mood Jam 2013, to achieve the exact mood of the song Iggy and The Stooges - Penetration.

Highlight: Canvas

Penetration is made by using only JavaScript and HTML5 (Canvas). The game is built on top of CreateJS, a framework that is programmed to support thousands of simultaneous sprites on-screen. Game is mobile optimized and runs 30fps on the first iPad.

jQuery for Unity - uQuery

uQuery is a port of jQuery 1.6 to Unity3D. Unity uses very flexible JScript.NET programming language, that has many familiarities to JavaScript. That's why many web developes feel natural to start coding with Unity.

I really love jQuery and I really love Unity. So why not to combine the two? As a weekend project, I started to port functionalities of jQuery to Unity

Highlight: Code translating

uQuery is not yet a complete port of actual jQuery, but it allows usage of many most frequently used jQuery commands. I started by reading and translating the code line by line and hacked workarounds when ever needed to.

Highlight: AJAX (PHP+JS)

Fixtop jQuery plugin

Fixtop is a simple jQuery plugin that gives any HTML element the ability to be always on the screen. If you try to scroll the "fixtopped" item away from the screen, it will be stacked to the top of the page.

Highlight: jQuery

jQuery is a great tool, because it allows others to built their own tools to extend the service. I'm always looking for ways to help my own work, as well as to help everyone else. That's why I love to do plugins and extensions, that can are easy-to-use.
